# facebook

A new Flutter project.# واجهة تطبيق فيسبوك لايت (Facebook Lite UI Clone)

## الوصف
تطبيق موبايل مبني باستخدام إطار العمل Flutter، يحاكي واجهة مستخدم تطبيق فيسبوك لايت . تم تطوير هذا المشروع لأغراض تعليمية ضمن مقرر تطبيقات الموبايل.

## المميزات
- الصفحة الرئيسية مع موجز للمنشورات.
- صفحة الأصدقاء.
- صفحة الرسائل.
- صفحة الفيديوهات.
- صفحة الإشعارات.
- شريط تنقل بين الشاشات.

## التقنيات المستخدمة
- Flutter
- Dart
- Material Design

## هيكلية المشروع
- شاشة الرئيسية
- شاشة الأصدقاء
- شاشة الرسائل
- شاشة الفيديوهات
- شاشة الإشعارات

## الهدف من المشروع
الهدف من هذا المشروع هو التدرب على تطوير تطبيقات Flutter، وتصميم واجهات المستخدم، والتنقل بين الشاشات، وبناء واجهات تطبيقات الموبايل.
